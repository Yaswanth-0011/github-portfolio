E-Commerce SQL Analysis (Microsoft SQL Server)

Project Overview

This project analyses an e-commerce dataset using Microsoft SQL Server.

The goal is to understand:

* Monthly revenue trends
* Top selling products
* Revenue by country
* Repeat customers
* High-value customers

Tools Used

* Microsoft SQL Server
* SQL

Files Included

* orders.csv → dataset
* ecommerce_analysis.sql → SQL queries
* advanced_analysis → SQL queries

Additional SQL Analysis

The project also includes an advanced SQL script containing additional business-focused queries such as:

* Total company revenue calculation
* Average order value analysis
* Revenue by product category
* Customer spending analysis
* Monthly order volume tracking
* Customer ranking using SQL window functions

These queries demonstrate deeper analytical skills and advanced SQL capabilities beyond the core business questions.
